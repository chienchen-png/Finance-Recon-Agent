# 用户上传数据库

> 👋 把 Excel 文件丢到这里，然后对智能体说 **"导入数据"**，剩下的全交给我。

## 操作步骤

1. 将你的依据文件（如 Q1-Q4 季度回款提成表）和目标文件（如销售提成汇总表）拖入此文件夹
2. 对智能体说：**"导入数据"**
3. 智能体会**首先检查身份是否已初始化**（若未初始化则引导完成执行人/审核人/操作编码设置）
4. 智能体会逐一询问每个文件的性质（依据 / 待核对），你只需点选即可
5. 导入完成后自动进行结构化扫描，然后就可以开始核对

## 文件夹说明

- 根目录：放待导入的 Excel 文件（.xls / .xlsx）
- `.已删除/`：存放你标记为"不需要"的文件（软删除，可恢复）
- 已导入的文件会自动重命名为 `[已导入]_原文件名.xlsx`，不会被重复导入

---

**支持格式**：`.xls`（旧版） / `.xlsx`（新版）
